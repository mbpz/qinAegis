qinAegis

本地优先的 AI 质量工程平台

专为前端 Web 项目设计的 TUI/CLI 测试工具。AI 驱动的测试用例生成、 本地沙箱执行、失败复盘和质量门禁 —— 一切尽在本地。

GitHub 查看文档
# 安装
brew install mbpz/qinAegis/qinAegis

# 初始化
qinAegis init

# 开始测试
qinAegis explore --url https://your-app.com

为什么选择 qinAegis？

从临时脚本到持久的本地质量知识库

🎯

AI 驱动的测试生成

基于视觉大模型理解页面，自动生成测试用例。无需维护 CSS selector，告别脆弱的测试脚本。

🔒

完全本地沙箱

基于 Playwright 的浏览器进程隔离，安全可靠。数据全部存储在本地 ~/.qinAegis/projects/，完全可控。

📊

四类测试覆盖

冒烟测试、功能测试、性能测试、压力测试，统一输出质量门禁结果，支持 CI 阻断。

🔍

智能失败复盘

截图、trace、console、network 全量证据采集，AI 辅助问题分类：产品 / 测试 / 环境 / 模型。

📁

测试资产治理

Draft → Reviewed → Approved → Flaky → Archived，完整的用例生命周期管理。

⚡

零依赖安装

brew 一键安装，无需 Docker、无需容器运行时。Playwright 自动管理浏览器实例。

技术架构

Rust + TypeScript，成熟开源项目集成

qinAegis CLI/TUI (Rust)
  ├─ Project / Case / Run / Gate Services
  ├─ Local FS Storage (~/.qinAegis/projects)
  └─ Sandbox Runtime (Playwright)
       ├─ Midscene.js — 视觉 act/assert/extract
       ├─ MCP-style Observer — 结构化页面状态
       ├─ Playwright — 稳定动作、trace、证据
       ├─ Lighthouse CI — 性能预算
       └─ k6 — 负载测试

AI Model (MiniMax-VL / Qwen3-VL / Ollama)
